## 🔥 今日头条：Anthropic 发布 Claude Opus 4.8，代码和推理能力小幅提升

Anthropic 今日发布 Claude Opus 4.8 模型，在代码、Agent 技能、推理和知识工作方面带来小幅但切实的提升，同时推出 Claude Code Dynamic Workflows 和 API effort 参数，为后续 Mythos 模型铺路。

- Opus 4.8 相比前代在长上下文推理、代码生成准确率上有可感知改进
- 新增 Claude Code Dynamic Workflows，允许开发者定义多步 Agent 流程
- API 新增 effort 参数，可控制推理计算量以平衡速度与质量
- 该版本被视为 Mythos 模型正式推出前的站台更新

> **编辑点评**：虽非 Mythos 首发，但 Opus 4.8 的增量更新表明 Anthropic 仍在加速迭代，同时为 API 用户提供更细粒度的推理控制。｜ ★★★★☆

## 📰 今日要闻：Anthropic 完成 650 亿美元 H 轮融资，估值 9650 亿美元

Anthropic 宣布完成 650 亿美元 H 轮融资，估值达 9650 亿美元，成为 AI 行业有史以来规模最大的单笔融资之一；本月早些时候，公司年化收入已突破 470 亿美元。

- 领投方包括 Altimeter Capital、Dragoneer、Greenoaks 和 Sequoia Capital
- 联合领投方包括 Capital Group、Coatue、D1 Capital Partners 等
- 资金将用于扩大模型训练与推理基础设施

> **编辑点评**：近万亿估值对应超过 470 亿年化收入，Anthropic 正在加速商业化，成为 OpenAI 最有力竞争者。｜ ★★★★★

## 📰 今日要闻：国家大基金领投 DeepSeek，首轮融资投前估值 450 亿美元

成立三年一直靠自有资金运转的 DeepSeek 开启首轮对外融资，由国家集成电路产业投资基金（大基金）领投，投前估值 450 亿美元，本轮融资规模约 700 亿元人民币。

- DeepSeek 自 2026 年成立以来以极致研发效率著称
- 该轮融资被视为中国 AI 行业分水岭事件
- 资金将用于扩大算力储备和模型迭代

> **编辑点评**：大基金入局意味着 DeepSeek 从自给自足转向国家资本赋能，中国 AI 竞争进入新阶段。｜ ★★★★★

## 📰 今日要闻：Meta 联合纽约大学发布 ATLAS，用 AI 自动翻译数学教材为 Lean 4 形式化代码

Meta 与纽约大学发布 ATLAS（Autoformalized Textbook Library At Scale），消耗 1830 亿 token，将 26 本数学教科书自动翻译为 Lean 4 形式化代码，验证了大规模自动形式化的可行性。

- 该项目是迄今为止规模最大的自动化数学形式化尝试
- 代码完全开源，可帮助数学定理的机械化证明
- 为 AI 在数学研究辅助领域的应用提供了关键基础设施

> **编辑点评**：1830 亿 token 的投入相当于用 AI 重写了整代数学教材，形式化验证的自动化门槛大幅降低。｜ ★★★★☆
