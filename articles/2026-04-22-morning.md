> 今日 AI 圈最大变量来自 OpenAI——ChatGPT Images 2.0 搭载 GPT Image 2 模型正式上线，支持联网搜索、单次生成 8 张连贯图像。与此同时，Anthropic 将 Claude Code 从 Pro 订阅中移除，定价策略出现明显分层。苹果方面，蒂姆·库克宣布将于 9 月卸任 CEO，约翰·特努斯接棒。

## 🔥 头条：ChatGPT Images 2.0 上线，生图模型首次具备"思考"能力

OpenAI 于 4 月 21 日正式发布 ChatGPT Images 2.0，基于 GPT Image 2 模型打造。本次升级的核心在于引入"思考能力"——用户启用后，系统可在生成前联网检索信息、解析上传文件、并对图像结构进行推理规划。这项功能目前已向 ChatGPT Plus、Pro、Business 及 Enterprise 订阅用户开放。

多图生成能力实现重大突破。Images 2.0 支持单次提示生成最多 8 张图像，并能在不同场景中保持角色和物体的一致性。Reddit 社区反馈显示，用户对 3×3 网格的生成质量已有显著感知提升，复杂场景的排版准确率大幅改善。有用户评价称，该模型在"review its own output and iterate until satisfied"能力上的表现尤为突出——生成后可自审并迭代修正。

从产品定位看，Images 2.0 标志着 OpenAI 在多模态赛道上的阶段性收网。图像生成从"一次性出图"演进为"推理—生成—自审"的闭环流程，这种范式变化对设计、广告、内容创作等下游场景的影响值得持续跟踪。

| 维度 | GPT Image 1 | GPT Image 2 |
|------|-------------|-------------|
| 单次生成上限 | 1 张 | 8 张 |
| 联网能力 | 无 | 支持实时检索 |
| 自审迭代 | 无 | 生成后自动修正 |
| 订阅覆盖 | Plus | Plus/Pro/Business/Enterprise |
| 角色一致性 | 弱 | 跨场景保持一致 |

> 来源：[OpenAI](https://openai.com/index/introducing-chatgpt-images-2-0/)（2026-04-21）｜ ★★★★★

> **编辑点评**：生图模型从"工具"升级为"有判断力的创作伙伴"，这个转变的行业意义大于技术参数本身。当模型开始在生成前检索信息、生成后自我修正，图像生成就不再是 prompt engineering 的问题，而是 agent 化工作流的起点。对于 Midjourney、Stable Diffusion 等竞品而言，单纯追求画质的赛道正在收窄。

## ★★★★☆ · Claude Code 从 Pro 订阅移除，Anthropic 定价策略出现分层

Anthropic 已将 Claude Code 从 Claude Pro 订阅的功能列表中移除。此前 Pro 用户可直接使用 Claude Code 进行代码编写，此次调整意味着 Claude Code 将成为更高阶订阅的专属功能。Hacker News 上该话题获得 298 分和 184 条评论，Reddit r/ClaudeAI 的相关 PSA 帖子获 912 upvotes 和 342 条评论，社区反应强烈。

同步值得关注的是，Anthropic 此前曾限制 OpenClaw 等第三方 CLI 工具对 Claude API 的调用，如今已恢复开放。这一系列动作表明 Anthropic 正在对不同用户群体进行精细化定价分层——免费/Pro 用户保留基础对话能力，企业/专业开发者通过更高价格获取高级编码工具和 API 灵活性。

> 来源：[Hacker News](https://claude.com/pricing)（2026-04-21）｜ ★★★★☆
> 来源：[Reddit r/ClaudeAI](https://reddit.com/r/ClaudeAI/comments/1srzhd7/psa_claude_pro_no_longer_lists_claude_code_as_an/)（2026-04-21）｜ ★★★★☆

> **编辑点评**：从商业逻辑看，Anthropic 对 Claude Code 的定价调整指向一个明确信号——AI 编码工具正从"通用功能"向"专业付费服务"演进。这对 Cursor、GitHub Copilot 等竞品的定价策略也将产生连锁反应。开发者工具的订阅成本正在成为企业 AI 预算中不可忽视的组成部分。

## ★★★★☆ · 奥尔特曼公开批评 Anthropic Mythos：称其"制造恐慌式营销"

OpenAI CEO 萨姆·奥尔特曼在《Core Memory》播客节目中公开批评 Anthropic 的 Mythos 安全模型，称 Anthropic 在利用恐慌情绪夸大产品实际能力。Mythos 于本月初发布，Anthropic 仅向少数企业客户开放试用，并声称该模型能力过强、不宜向公众开放，担心被网络犯罪分子利用。

批评人士认为 Anthropic 的说法言过其实。奥尔特曼在节目中直言，这种"制造恐慌式营销"的目的是让 AI 技术只掌握在少数公司手中。这是两家头部 AI 公司近期最直接的公开对峙，也折射出安全赛道上的竞争正在从技术层面延伸到叙事权的争夺。

> 来源：[IT之家](https://www.ithome.com/0/941/860.htm)（2026-04-21）｜ ★★★★☆

> **编辑点评**：AI 安全已经成为竞争工具而非纯粹的技术议题。Anthropic 通过"能力过强不便公开"的叙事建立品牌护城河，OpenAI 则以"反垄断"话术回击。这场口水战的背后，是企业级安全市场的蛋糕分配之争。对用户而言，关注点应从谁喊得更响转向谁的安全方案真正可验证、可审计。
