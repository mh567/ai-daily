## 🔥 今日头条：GPT-5.6 正式发布，OpenAI 重回最强基模王座

OpenAI 于今日正式发布 GPT-5.6 模型，性能超越此前在多项基准上领先的 Fable5，重新夺回最强基础模型地位。但模型能力逼近关键阈值后，使用资格与方式可能成为比性能更受关注的事项。

- GPT-5.6 在多项基准测试中大幅刷新纪录，具体性能数据未在官方发布中完整披露
- 由于模型逼近关键敏感能力阈值，OpenAI 可能对 API 和网页端使用施加更严格的资格限制
- 目前普通用户尚无法通过 ChatGPT 直接访问 GPT-5.6，部分测评者已获得预览权限

> **编辑点评**：能力越强、管制越紧，GPT-5.6 的“可用性”或许比“能力值”更能决定其实际影响。｜ ★★★★★

## 📰 今日要闻：DeepSeek 联合北大开源 DSpark 推理加速框架

DeepSeek 与北京大学今日联合发布 DSpark 推理加速框架，通过缓存与推测解码优化，在同等吞吐量下将单用户生成速度提升 60% 至 85%，已部署于 DeepSeek-V4-Flash 与 DeepSeek-V4-Pro 预览版服务引擎中。

- DSpark 采用单 token 推测解码基线 MTP-1 作为对比，在高并发场景下实现显著延迟降低
- 相关论文、训练代码已在 GitHub 上开源，开发者可直接复现或适配
- 该框架针对大语言模型自回归生成时的“每 token 一次前向传播”瓶颈进行优化

> **编辑点评**：推理效率开源项目直击落地成本痛点，有望降低高端模型的生产环境部署门槛。｜ ★★★★☆

## 📰 今日要闻：阿里千问输入法 macOS 版上线，支持最快 300 字/分语音输入

阿里千问输入法今日正式上线 macOS 版，支持最快 300 字/分钟语音输入，AI 自动润色，并可识别 9 种方言。官网显示 iOS 版、Android 版、Windows 版将于近日发布。

- 产品定位“说完即成稿”，语音转文字后自动纠错、润色
- 该输入法不含广告，PC 端侧重语音输入，移动端将适配 AI 键盘功能
- 此前千问已推出“千问语音输入法”网页版，本次为独立桌面应用

> **编辑点评**：千问输入法补齐了阿里在 AI 输入法赛道的桌面端拼图，与竞品展开正面竞争。｜ ★★★★☆

## 📰 简讯：高通计划将数据中心芯片堆叠架构引入手机 SoC

高通执行副总裁杜尔加·马拉迪透露，公司计划将数据中心用高带宽计算（High Bandwidth Compute）架构——通过垂直堆叠芯片让内存与计算芯片更靠近——引入智能手机 SoC，以提升端侧 AI 性能。第一代数据中心产品预计明年量产。
