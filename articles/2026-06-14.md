## 🔥 今日头条：OpenAI GPT-5.5 与 Codex 正式登陆 Amazon Bedrock

OpenAI 最新模型 GPT-5.5 及编程辅助模型 Codex 正式在 Amazon Bedrock 平台上架，企业用户可通过 AWS 直接调用。

- 此次上架的是 GPT-5.5 系列模型，性能相比前代有显著提升
- Codex 专门面向代码生成与理解场景，支持多语言编程
- 企业可通过 Bedrock 的 API 按需调用，无需自行部署硬件
