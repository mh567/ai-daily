## 🔥 今日头条：字节火山引擎发布业界首个“Agent套餐包”，40元/月撬动多模态AI能力

火山引擎正式发布 Agent Plan，这是业界首个以“Agent”命名的订阅套餐包，相比此前的 Coding Plan 新增更多模态模型和 Harness 工具。

- Agent Plan 包含 Doubao-Seed、Doubao-Seedance、Doubao-Seedream 等字节跳动系列 SOTA 模型，以及 GLM-5.1、Kimi-K2.6 等主流三方模型
- 配套提供联网搜索等 Harness 工具，满足 Agent 场景更全面的需求
- 官方表示未来将持续更新模型和工具，每月 40 元起

> **编辑点评**：字节将模型、工具打包为一个完整 Agent 订阅服务，降低了开发者和企业搭建自主 Agent 的门槛，这是国内云计算巨头在 Agent 基础设施层的重要卡位。｜ ★★★★★

## 📰 今日要闻：阿里发布AI店小蜜，“AI+人”协同转化效果相较纯人工显著提升

阿里今日发布全新 AI 店小蜜，宣称是电商行业首个具备售前售后办事能力的客服 agent。

- 实测数据显示商家接入后平均转人工率下降 45%，“AI+人”协同转化效果相比纯人工客服增加超 10%，首次实现人工协同转化效果提升
- 全链路基于千问最新大模型构建，依托淘宝交易数据，针对不同行业进行了垂域微调与多模态升级
- 截至 3 月，AI 店小蜜日均对话量近千万，接入商家数突破百万

> **编辑点评**：这是 AI 客服能力从“替代人工”迈向“协同增效”的重要节点，低成本高效率的智能导购将在 618 大促中释放显著商业价值。｜ ★★★★☆

> 来源：[IT之家](https://www.ithome.com/0/948/909.htm)（2026年5月11日）

## 📰 今日要闻：支付宝推“AI低价帮抢”，千问与淘宝打通后的新支付场景

支付宝“AI付”今日发布“AI 低价帮抢”功能，用户可在淘宝 App 中通过千问 AI 购物助手设置目标，由 AI 完成蹲价和下订单。

- 支持完成 1000 元内商品的购买，三步完成：打开千问助手 → 输入需求 → 完成核身授权
- 支付采用“一次指令、一次授权、用完失效”的委托代办模式
- 这是千问与淘宝全面打通后，AI 从信息获取延伸到实际交易的闭环尝试

> **编辑点评**：AI 代买的合规性与安全性是最大挑战，支付宝明确限定 1000 元和一次性授权，在实验阶段保持了控制边界。｜ ★★★★☆
