## 🔥 今日头条：Opus 4.8 基准测试领先 GPT-5.5，沃顿教授用其开发全球投胎模拟器

Claude Opus 4.8 在多项基准测试中领先于 GPT-5.5 登顶，沃顿商学院教授 Ethan Mollick 利用该模型独立开发「The Veil of History」网站，基于人口统计学数据模拟人类投胎概率。

- Opus 4.8 在推理、编程、数学等基准测试中领先于 GPT-5.5，成为当前最强闭源模型之一
- Ethan Mollick 使用 Opus 4.8 独立完成全栈开发，从数据处理到前端部署均依赖大模型完成
- 「The Veil of History」网站允许用户输入出生年份、性别等参数，模拟随机投胎至全球不同地区和历史时期

> **编辑点评**：Opus 4.8 的基准表现和大模型辅助开发的能力同时得到验证，预示 AI 正从“聊天工具”向“自主构建者”演变。｜ ★★★★★

## 📰 今日要闻：OpenAI 与日本云服务商 Data Section 达成战略合作，抢攻亚太企业级 AI 市场

日本云服务提供商 Data Section 宣布与 OpenAI 达成战略合作，通过其平台为亚太地区客户提供经监管合规的 OpenAI 模型。

- Data Section 在日本、泰国、马来西亚、澳大利亚和美国部署了英伟达 GPU 集群数据中心
- 其企业级 AI 工作流平台 TAIZA 将在符合当地监管要求的框架下接入 OpenAI 模型
- 此次合作标志着 OpenAI 从依赖微软 Azure 的单平台策略转向多区域分布式算力部署

> **编辑点评**：OpenAI 通过第三方云平台分散基础设施风险，同时借力本地服务商解决亚太市场的合规和落地问题，是全球化战略的关键一步。｜ ★★★★☆

## 📰 今日要闻：英伟达清华团队提出 Gamma-World，世界模型从单人交互升级为多人共处

英伟达与清华大学联合团队提出 Gamma-World，首次实现多个智能体在同一世界模型中同时交互、协同感知与行动。

- Gamma-World 允许多个用户/智能体同时参与同一个 3D 虚拟世界的物理模拟与角色控制
- 技术建立在扩散世界模型基础上，通过空间分解和时间同步实现多人实时共存
- 该成果可用于游戏、虚拟训练、多智能体协作等场景，是具身智能的重要一步

> **编辑点评**：从“一个人玩”到“多人共处”，世界模型迈入多智能体协同时代，为混合现实和机器人训练开辟了新路径。｜ ★★★★☆

## 📰 简讯：Hermes Agent 加入工具搜索，显著节省 tokens 开销

Nous Research 为开源 Hermes Agent 加入 Tool Search 功能，按需加载工具以缓解 MCP 协议占用上下文的问题。在含 5 台 MCP 服务器和 34 个工具的部署中，平均每回合约 50% 的 tokens 原用于工具模式描述，新方案可大幅降低开销。
