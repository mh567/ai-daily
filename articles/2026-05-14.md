## 🔥 今日头条：Anthropic 推出中小企业专属 Claude 服务，AI 竞争下沉至小微商户

Anthropic 宣布推出 Claude for Small Business 服务套餐，专为中小企业和小微商户定制，将 AI 能力从大型企业扩展至社区级商户。

- 该套餐提供针对工资规划、月末结算、发票催收、客户跟进等小企业常见场景的模板与工具
- 区别于面向大型企业的复杂方案，Claude for Small Business 强调低部署成本和易用性
- 此次发布标志着 Anthropic 正式进入中小企业市场，与 OpenAI 等对手展开更广泛竞争

> **编辑点评**：企业级 AI 落地长期集中在预算充足的巨头，Anthropic 此举瞄准了体量最大但最被忽视的小微市场，可能加速 AI 在各行业的普惠化进程。｜ ★★★★★

## 📰 今日要闻：腾讯开源 Agent 记忆技术方案，Token 消耗最高降低 61%

腾讯云数据库团队正式开源 TencentDB Agent Memory，一套面向 AI Agent 的分层记忆引擎，采用 MIT 协议，旨在解决 Agent 长会话中的上下文保留和记忆复用问题。

- 超长 session 评测显示，该方案可降低最高 61% 的 Token 消耗
- 支持符号化记忆与分层记忆架构，适用于 Agent 工作流学习与跨会话经验沉淀
- 项目已在 GitHub 开源，可开箱即用

> **编辑点评**：Token 成本是 Agent 规模化落地的关键瓶颈，腾讯从记忆层切入，开源方案或推动行业节约推理成本。｜ ★★★★☆

> 来源：[量子位](https://www.qbitai.com/2026/05/417753.html)（2026年5月14日）
