## 🔥 今日头条：Cloudflare 正式推出 Agent Memory，为 AI 智能体提供持久记忆托管服务

Cloudflare 宣布上线 Agent Memory 服务，旨在为 AI 智能体（Agent）提供可持久化的记忆存储与检索能力，解决当前大模型对话上下文受限、智能体无法跨会话保持记忆的痛点。

- 该服务允许智能体将关键状态、用户偏好、任务进度等数据存储在 Cloudflare 边缘网络，实现低延迟访问。
- 开发者通过简单的 API 即可集成，无需自建记忆系统，支持与 Cloudflare Workers 和 AI Gateway 配合使用。
- Cloudflare 表示，Agent Memory 旨在成为 AI 智能体的“长期记忆层”，可提升智能体在复杂任务中的连贯性和个性化体验。

> **编辑点评**：智能体持久化记忆是当前 AI 应用落地的关键瓶颈之一。Cloudflare 以边缘平台切入，提供了低成本、高可用的托管方案，可能加速智能体在企业场景的部署。｜ ★★★★★

## 📰 今日要闻：中国机器人创企灵心巧手寻求 60 亿美元估值

灵心巧手宣布在下一轮融资中目标估值 60 亿美元，较刚完成的 B+ 轮融资估值（30 亿美元）翻倍。该公司专注于高灵巧度人形机器人机械手，成立两年即成为独角兽，早期投资方包括蚂蚁集团和红杉中国，最新一轮有国资背景基金参与。

- 公司上周完成 B+ 轮融资，估值达 30 亿美元，具体融资金额未披露。
- 目标估值 60 亿美元，对应下一轮私募融资或 IPO，公司未明确时间表。
- 灵心巧手称是全球人形机器人高灵巧度机械手市场的领先企业，客户涵盖多家头部人形机器人整机厂商。

> **编辑点评**：机器人灵巧手是具身智能的关键零部件，灵心巧手估值快速翻倍反映出资本市场对具身智能供应链的追捧，但 60 亿美元目标能否达成仍存变数。｜ ★★★★☆

## 📰 今日要闻：“DeepSeek版 Claude Code”开源，GitHub 星标超 2,300

一款名为“DeepSeek版 Claude Code”的开源命令行工具在 GitHub 上获得 2,300 多颗星标。该项目模仿 Claude Code 的交互方式，但后端接入 DeepSeek 模型，允许开发者在终端中直接与 DeepSeek 对话、执行代码操作。

- 项目提供类似 Claude Code 的终端 UI，支持自然语言调用 DeepSeek API。
- 开发者可自定义模型参数，适用于代码生成、调试、文件操作等场景。
- 作者称该项目旨在降低国内开发者使用 AI 编程助手的门槛，无需代理即可直连 DeepSeek。

> **编辑点评**：Claude Code 是 Anthropic 近期推出的热门编程工具，DeepSeek 版本的快速克隆表明开源社区对大模型替代品的旺盛需求，也侧面验证了 Claude Code 交互范式的受欢迎程度。｜ ★★★★☆

## 📰 今日要闻：GitLab 新增固定费率代码审查服务及 Free-Tier AI 访问

GitLab 宣布推出采用固定费率的代码审查服务，并开放 Free-Tier 层级的 AI 访问权限，同时引入支出上限设置功能。新服务旨在降低团队使用 AI 辅助代码审查的成本不确定性。

- 固定费率代码审查服务面向企业用户，按月或按年收取固定费用，不依赖 token 消耗量。
- Free-Tier 用户现在可以有限度地使用 GitLab Duo AI 功能，无需付费。
- 新增支出上限设置，帮助用户控制 AI 相关服务的预算。

> **编辑点评**：GitLab 将 AI 辅助代码审查从按量计费转为固定费率，降低了企业使用门槛，同时 Free-Tier 开放有助于培养用户习惯，是争夺 AI 开发者工具市场的明智之举。｜ ★★★★☆
