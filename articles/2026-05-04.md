## 🔥 今日头条：Cloudflare 推出 Agent Memory，云平台开始接管智能体长期记忆

Cloudflare 推出面向 AI 智能体的托管服务 Agent Memory，并开启私测。它把智能体对话中的用户偏好、任务上下文和历史状态提取成结构化记忆，再通过托管接口供智能体后续调用。

这项服务针对的是智能体应用的一个基础问题：模型单次上下文窗口可以保存短期信息，跨会话、跨任务的稳定记忆仍需要外部存储和检索系统。Cloudflare 的方案把这部分能力下沉到基础设施层，和 Workers、AI Gateway 等边缘与 AI 服务形成组合。

InfoQ 报道称，Agent Memory 支持从智能体对话中提取结构化记忆，使用五通道并行检索与倒数排名融合技术按需调取，还支持智能体团队共享记忆知识库。

> **编辑点评**：智能体走向生产环境后，记忆会成为和向量数据库、权限控制、工具调用同等级的基础设施能力。Cloudflare 的切入点清晰：让开发者不用为每个智能体应用重复搭建记忆层，同时把记忆访问放到边缘网络和现有开发者生态中。｜ ★★★★★

> 来源：[InfoQ](https://www.infoq.cn/article/TPqCEvSNCh9jzivioLs8?utm_source=rss&utm_medium=article)（5月4日）

## 📰 今日要闻：中国机器人创企灵心巧手寻求新一轮融资，目标估值 60 亿美元

中国机器人公司灵心巧手正在寻求下一轮融资，目标估值为 60 亿美元。IT之家援引路透社报道称，灵心巧手 CEO 周永表示，公司目前占据全球高自由度机械手市场超过 80% 份额，并计划很快把月产能从接近 5000 台提升至 1 万台。

灵心巧手的核心产品是用于人形机器人和工业机器人的高灵巧度机械手。随着具身智能和人形机器人产业链升温，灵巧手成为上游关键部件之一，也成为资本重点关注的环节。

> **编辑点评**：灵巧手既是人形机器人商业化的技术门槛，也是供应链议价能力较强的零部件。灵心巧手用产能、份额和融资预期同时讲增长故事，后续需要用稳定交付和客户结构来支撑高估值。｜ ★★★★☆

## 📰 今日要闻：“DeepSeek 版 Claude Code”开源，国内开发者复制终端 AI 编程范式

量子位报道，一款面向 DeepSeek 优化、交互形式类似 Claude Code 的开源命令行工具在 GitHub 获得 2.3k 星标。项目把大模型能力放进终端交互流程，开发者可以在命令行里用自然语言发起代码理解、修改、调试等任务。

这类项目的重点不只在模型调用，也在交互范式：终端内对话、文件上下文读取、代码编辑建议和任务执行反馈构成了新的 AI 编程工作流。DeepSeek 生态的开源实现降低了国内开发者试用同类体验的门槛。

> **编辑点评**：Claude Code 让“终端里的编程智能体”成为明确产品形态，开源社区迅速用本土模型复刻，说明开发者工具的竞争正在从模型能力延伸到工作流设计、权限边界和可控执行体验。｜ ★★★★☆
